# LIGA-ADMIN - Tournament Management Dashboard

React 19 admin interface for managing competitive seasons.

## Quick Start

```bash
# Install dependencies
npm install

# Configure environment
cp .env.example .env
# Edit .env with your Supabase credentials

# Start dev server
npm run dev
```
