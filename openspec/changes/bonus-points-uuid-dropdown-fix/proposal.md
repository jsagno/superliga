# Propuesta: Bonus Points UUID y Dropdown

## Problemas
1. Error al guardar bonus-points: el id generado no es UUID válido, lo que causa un error en backend.
2. El dropdown de selección de jugador no tiene los estilos estándar de la app.

## Objetivo
- Garantizar que todos los bonus-points usen UUID v4 válidos como id.
- Unificar el estilo visual del dropdown con el resto de la app.

## Justificación
- Mejora la integridad de datos y la experiencia de usuario.
- Evita errores de guardado y confusión visual.
